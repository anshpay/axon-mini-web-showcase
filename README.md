# AXON Mini Landing Page

Modern product landing page for the AXON Mini edge AI board.

## Run locally

```bash
python -m http.server 4173
```

Open: `http://127.0.0.1:4173/index.html`

## Files

- `index.html` - page structure
- `styles.css` - visual design and responsive layout
- `main.js` - Three.js scene and scroll choreography
- `assets/` - extracted and cleaned product image assets
